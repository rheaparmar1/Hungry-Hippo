
Instructions/Hints
 
  Guide your game piece (the hippo) with the arrow keys to the fruit. Each fruit collected results in a increases of your points. Achieve a total of 5 points to move on to the next level.
  Make sure you avoid hitting bombs or the game border, doing so results in a loss. Ensure to maneuver around sand patches though if used strategically be used to slow down the game in 
  difficult levels.

	- Skip levels using th level menu bar
	- Turn of music/sound effects in Settings 
	- Manually start a new game (and aditional game settings) in the Game bar
	- Both A, W, S, D or arrow keys can be used


Added Functionalities
	- Speed increases based on level 
	- Obstacles (sand/walls) placed at random in later levels
